# Service Oriented Architecture (SOA)


La tendencia de desarrollo de software utiliza arquitecturas de software orientadas a servicios (SOA), las cuales brindan:

- eficiencia, 
- agilidad y 
- facilidad de crecimiento. 

El diseño arquitectónico más utilizado en el desarrollo de aplicaciones SOA es REST (Representational State Transfer); sin embargo, se han identificado algunos problemas de gestión de datos en su Interfaz de Programación de Aplicaciones llamada API-REST. Han surgido varias opciones tecnológicas para apaciguar estos problemas, como:

- SPARQL, 
- Cypher, 
- Gremlin y la más popular 
- GraphQL. 

GraphQL fue desarrollado por Facebook en 2012 y lanzado en 2015 a la comunidad como un proyecto de código abierto, utilizado por empresas como:

- GitHub, 
- Airbnb, 
- Amazon, 
- Apollo, 
- IBM y 
- Facebook.
